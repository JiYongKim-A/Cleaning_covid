## 🎬 어떤 프로젝트 인가요?

 ‘Cleaning covid-19’는 pygame 모듈을 이용하여 만든 2D 슈팅 게임 입니다.
 pygame 모듈을 통해 2D 게임을 만드는 영상을 응용하여 간호사가 바이러스를 제거하며 Score를 얻어 기록할 수 있는 간단한 2D 게임을 구성해 보았습니다.
